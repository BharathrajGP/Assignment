We have 3 types of CSS they are

  1. Inline CSS :
    • Which is applied using style attribute inside a tag
    • It is limited only for that tag
  2. Internal CSS :
    • Which is applied using style tag inside a head section
    • It is limited only for that that file
  3. External CSS :
    • Which is applied by creating external .css extension file and linking it to html file by using link tag in head section
    • We can use this properties for any of html file by linking css file location in link tag

CSS Selectors

  • CSS selectors are used to select the HTML elements based on tag name ,class name,id name etc.
  • There are 7 types of selectors the are
    1. Element Selector :Select only one type of element based on tag name
    2. Group Selector :Select two or more elements at a time
    3. Class Selector :Selects based on class name but only works on specific tag for which class is specified
    4. Generic class Selector :Selects based on class name and available for all tags
    5. ID Selector :Selects based on ID name and available for all tags
    6. Child Selector :Selects only the child elements
    7. Universal Selector :Selects all tags
