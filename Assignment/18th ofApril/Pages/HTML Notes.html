HTML


Introduction

  • HyperText Markup Language
  • Introduced by Tim Berners-Lee in 1993
  • By using HTML we can create static web pages
  • Latest version of HTML is HTML5

Tag

  • Word or letter surrounded by angle brackets is called tags
  • We have two types of tags they are

  1. Paired tag :Which consist of opening and closing tag are called paired tags
  2. Unpaired tag : Are those which don't have any type of closing tags

HTML Element

  • A content along with opening and closing tag is known as HTML Element
